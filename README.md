# Enterprise Network Simulation (CCN)

## 📌 Project Overview
This repository contains a comprehensive network topology simulation designed in Cisco Packet Tracer (`raza sher project ccn.pkt`). The project demonstrates the practical application of Computer Communication Networks (CCN) principles, focusing on enterprise-grade network design, subnetting, and the configuration of essential network services and security protocols.

## 🚀 Key Features & Configurations
The simulation includes the implementation and configuration of several critical network protocols and services:
* **Dynamic Host Configuration Protocol (DHCP):** Automated IP address allocation across multiple subnets.
* **Domain Name System (DNS):** Hostname resolution for simulated internal web and file servers.
* **Secure Shell (SSH) & Telnet:** Configuration of secure and standard remote access protocols for network administration.
* **Routing & Switching:** Inter-VLAN routing and optimized switch configurations for seamless communication across different network segments.
* **Access Control & Security:** Basic network security measures and administrative access controls.

## 🛠️ Tools & Technologies Used
* **Simulation Software:** Cisco Packet Tracer
* **Core Domains:** Network Administration, Network Security, Subnetting, Routing & Switching
